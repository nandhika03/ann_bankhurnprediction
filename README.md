# Bank Chrun Prediction using ANNs

Churn is when a customer cancels, stops using, or switches away from a service.

Churn rate = the percentage of customers who leave over a certain period.
Churn Rate = Number of customers lost during a period / Total number of customers at the start of the period × 100

Losing customers is expensive and acquiring new ones costs more than retaining existing ones. Predicting who is likely to churn allows proactive retention strategies:

- Offers
- Discounts
- Personalized communication

Input: customer data (age, balance, geography, etc.)
Output: whether the customer will churn or not (binary classification: 1 = churn, 0 = stay)
